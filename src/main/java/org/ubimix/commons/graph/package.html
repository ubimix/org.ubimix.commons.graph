This package contains classes used to iterate over graph structures.

The basic class for all other structures is the Walker class. It translates simple calls to the #begin(Object)/#end() methods to corresponding calls of the IWalkerListener class. Walker instances are used internally by the following high-level classes:

  • GraphIterator/IteratorBasedGraphIterator - these classes are used to iterate over graph structures.
  • TreeBuilder - this class is used to transform a sequence of paths into the corresponding graph structure.
